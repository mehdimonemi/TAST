IBM ILOG CPLEX Optimizers 12.6.2

This directory includes the CPLEX Optimizers of IBM® ILOG® CPLEX® Optimization Studio 12.6.2, including Concert Technology for C++, Java and .NET users. Please read the CPLEX Optimization Studio 12.6.2 Release Notes in order to take full advantage of the new features in this release and to ensure a smooth transition from prior versions of CPLEX.

Folder Structure

The following folders are contained in the folder where you installed CPLEX Optimization Studio 12.6.2.

In the cplex folder:

    bin\x86_win32 or bin\x64_win64:
	Executable files, assemblies and assembly documentation, including:
	  cplex.exe - CPLEX Interactive Optimizer
	  cplex1262.dll - CPLEX Callable Library
	  cplex1262remote.dll - CPLEX remote object library
	  cplex1262remotejni.dll - JNI bindings for the remote object
	  cplex1262<y>worker.dll - Remote object worker library for protocol <y> 
	  cplex1262<y>transport.dll - Remote object transport library for protocol <y> 
	  ILOG.CPLEX.dll and ILOG.Concert.dll - Concert Technology .NET assemblies
	  cplexamp.exe - Solver driver to use with AMPL
	  convert.exe - Conversion utility for files written in some older formats

    examples\src\<language>:
	Source files for the examples (lpex1.c is in examples\src\c, et cetera).

    examples\<compiler>\<libformat>:
	Visual Studio solution and project files for building the C, C++, C# and VB examples
	to support your particular x86_win32 or x64_win64 platform and and an nmake makefile,
	javamake, for building the Java examples.

    excel:  
	Files for the CPLEX add-in for Excel.  Instructions for installing 
	the add-in are in the CPLEX for Microsoft Excel User's Manual.

    include\ilcplex:
	Files, such as cplex.h and ilocplex.h, that are needed to compile applications
	that use the Callable Library or the C++ version of Concert Technology.

    lib\<compiler>\<libformat>:
	Library files that contain CPLEX (cplex1262.lib) and
	CPLEX extensions to the C++ version of Concert Technology (ilocplex.lib).

    lib:
	The cplex.jar file containing the Java classes needed to use the Java version
	of Concert Technology.

    matlab\x86_win32 or matlab\x64_win64 :
	Files comprising the CPLEX for MATLAB interface.  
	Refer to the section "Setting up CPLEX for MATLAB" of "Getting started with CPLEX for MATLAB"
	for more information on configuiring your environment.

    python\<version>\x86_win32 or python\<version>\x64_win64:
	Files comprising the Python interface for CPLEX.  Several versions may be provided.
	Some setup.py scripts which run distutils are included.

In the concert folder:

    include\ilconcert:
	Files, such as ilomodel.h, that are needed to compile applications that use the
	C++ version of Concert Technology.

    lib\<compiler>\<libformat>:
	Concert Technology libraries for C++ users (concert.lib).

Compiling and Executing

To compile and execute the C or C++ examples, or to write a new project using CPLEX from C or C++, please consult the file c_cpp.html. To compile and execute the C# or Visual Basic.NET examples, or to write a new project from C# or Visual Basic.NET, please consult the file dotnet.html.

To compile and execute the Java examples, a makefile for use with Microsoft's nmake is provided in each examples\<compiler>\<libformat> folder.

Type nmake -f javamake to compile all the examples.
Type nmake -f javamake execute to run all the examples.

The file javamake also indicates how to set the location of the CPLEX shared library and cplex.jar file used from Java.
